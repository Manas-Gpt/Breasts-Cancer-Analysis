About BreastGuard AI

BreastGuard AI is an educational AI-assisted mammography analysis system inspired by the NYU Breast Cancer Classifier (Shen et al., 2019). It classifies mammograms into three categories using deep learning-inspired feature analysis.

✅
Normal

No suspicious findings. Routine annual screening recommended.

⚠️
Benign

Non-cancerous findings present. Follow-up imaging in 6 months advised.

🚨
Malignant

Potentially cancerous findings detected. Immediate oncology consultation required.

🔬

Technical Details

Model Random Forest + Calibrated Probabilities
Training Data Wisconsin Diagnostic Breast Cancer (UCI)
Views Supported L-CC, L-MLO, R-CC, R-MLO
Features 30 radiometric image descriptors
Reference NYU Breast Cancer Classifier (2019)
Backend Python · Flask · scikit-learn
Report Format PDF via ReportLab
